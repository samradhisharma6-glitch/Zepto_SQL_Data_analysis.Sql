🛒 Zepto SQL Data Analysis Project
📌 Project Overview

This project analyzes Zepto's e-commerce inventory dataset using SQL to uncover business insights related to product pricing, inventory management, discounts, and category performance.

The objective of this project is to demonstrate SQL data analysis skills by performing data cleaning, exploratory data analysis (EDA), and business-driven analysis on a real-world retail dataset

🎯 Business Objectives
Analyze product pricing and discount patterns.
Identify high-value and low-value product categories.
Examine inventory availability across products.
Discover top-performing categories based on pricing and discounts.
Generate actionable insights for inventory and pricing strategies.
